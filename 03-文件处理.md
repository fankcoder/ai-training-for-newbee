# 模块三：文件处理

> 🎯 目标：掌握AI处理各类办公文件的能力，包括文档生成、格式转换、内容编辑等

---

## 3.1 AI文件处理能力概览

### 支持的文件类型

| 类型 | 能力 | 常用场景 |
|------|------|---------|
| Word (.docx) | 创建、编辑、格式化 | 报告、合同、方案 |
| Excel (.xlsx) | 创建、公式、图表、数据分析 | 报表、预算、统计 |
| PDF (.pdf) | 读取、提取文字、合并、拆分 | 合同、报告、手册 |
| PowerPoint (.pptx) | 创建、编辑幻灯片 | 汇报、培训、演示 |
| 文本 (.txt/.md) | 创建、编辑、转换 | 笔记、配置、文档 |
| CSV (.csv) | 读取、转换、分析 | 数据导入导出 |

## 3.2 Word文档处理

### 创建文档

```
请帮我创建一份项目立项报告，包含以下章节：
1. 项目背景
2. 项目目标
3. 实施计划
4. 预算估算
5. 风险评估

要求：
- 使用正式商务风格
- 每章节300-500字
- 留有数据填充位置
```

### 编辑优化

```
请帮我优化这份文档：
1. 修正错别字和语法错误
2. 统一格式和用词
3. 增强逻辑连贯性
4. 添加目录和页眉
```

### 格式转换

```
请将以下Markdown内容转换为Word文档格式：
[粘贴内容]
```

## 3.3 Excel表格处理

### 创建表格

```
请帮我创建一个员工信息登记表，包含以下列：
- 员工编号（自动编号）
- 姓名
- 部门
- 职位
- 入职日期
- 联系电话
- 邮箱

要求：
- 第一行作为表头并加粗
- 设置数据验证（部门只能选：技术部/销售部/人事部/财务部）
- 入职日期格式为YYYY-MM-DD
```

### 公式计算

```
请为以下销售数据表添加公式：
1. D列=单价×数量（计算销售额）
2. 最后一行添加合计
3. 计算平均销售额
4. 标注高于平均值的记录
```

### 数据分析

```
这是我最近3个月销售数据，帮我分析问题和改进建议：
1. 按部门汇总销售额
2. 找出Top 5销售产品
3. 计算各月环比增长率
4. 生成分析结论
```

## 3.4 PDF文档处理

### 内容提取

```
请提取以下PDF文档的核心内容：
1. 文档主题
2. 关键数据
3. 重要结论
4. 行动建议
```

### 文档合并

```
请将以下3个PDF文件合并为一个：
1. 合同正文
2. 附件A：技术参数
3. 附件B：服务条款
```

### PDF转文字

```
请将这份PDF文件转换为可编辑的文本格式，保留原有段落结构
```

## 3.5 实操案例

### 案例1：自动生成会议纪要

```
请根据以下会议记录创建一份正式的会议纪要Word文档：

参会人员：张三、李四、王五
会议时间：2026年4月15日
会议内容：
- 张三汇报了项目进度，目前完成60%
- 李四提出需要增加2名开发人员
- 王五确认了预算审批流程

要求：
- 标准会议纪要格式
- 包含决议事项和责任人
- 待办事项列表
```

### 案例2：数据报表生成

```
请根据以下原始数据创建Excel报表：

原始数据：
产品A：1月100件，2月120件，3月150件
产品B：1月80件，2月90件，3月110件
产品C：1月60件，2月70件，3月85件

要求：
1. 创建数据表格
2. 计算各产品季度总量
3. 计算月度环比增长率
4. 生成季度销售趋势折线图
5. 添加数据透视表
```

### 案例3：合同模板生成

```
请帮我创建一份标准的服务合同模板，包含：
- 甲乙双方信息
- 服务内容和标准
- 费用和支付方式
- 保密条款
- 违约责任
- 争议解决
- 签字盖章区域

要求：留出可填写的空白字段
```

## 3.7 效率提升技巧

1. **模板复用**：让AI生成模板，以后直接填写
2. **批量处理**：多个类似文件一次性让AI处理
3. **标准规范**：提前定义好格式要求，保持一致性
4. **检查验证**：AI生成后务必人工检查关键数据和公式
