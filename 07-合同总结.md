# 模块七：合同智能总结与分析

> 🎯 目标：掌握使用AI进行合同审查、风险识别、要点提取和智能总结的专业能力

---

## 7.1 合同处理能力概览

### AI在合同场景的能力

| 能力 | 说明 | 价值 |
|------|------|------|
| 要点提取 | 快速识别合同核心条款 | 节省90%阅读时间 |
| 风险分析 | 识别潜在风险和不利条款 | 降低法律风险 |
| 条款对比 | 对比不同版本或模板 | 快速发现差异 |
| 合规检查 | 检查是否符合法规要求 | 提高合规性 |
| 语言优化 | 优化合同表述 | 减少歧义 |
| 摘要生成 | 生成合同执行摘要 | 快速决策 |

## 7.2 合同审查流程

### 标准审查流程

```
[合同原文] 
    ↓
① [基本信息提取]
   - 合同类型
   - 当事方
   - 签署日期
   - 有效期
    ↓
② [核心条款分析]
   - 标的和范围
   - 价格和支付
   - 交付和验收
   - 违约责任
   - 争议解决
    ↓
③ [风险识别]
   - 法律风险
   - 商业风险
   - 操作风险
    ↓
④ [建议输出]
   - 修改建议
   - 谈判要点
   - 注意事项
```

## 7.3 合同要点提取

### 基本信息提取

```
请提取以下合同的基本信息：

1. 合同类型：
2. 甲方（委托方/买方）：
3. 乙方（受托方/卖方）：
4. 签署日期：
5. 合同有效期：
6. 合同金额：
7. 付款方式：
8. 交付时间：
```

### 核心条款提取

```
请提取以下合同的核心条款：

1. 【服务范围】具体包括什么？
2. 【付款条件】付款节点和比例？
3. 【交付标准】如何验收？
4. 【保密义务】保密范围和期限？
5. 【违约责任】违约金和赔偿？
6. 【知识产权】归属和使用？
7. 【争议解决】管辖地和方式？
8. 【终止条件】什么情况下可以终止？
```

## 7.4 合同风险分析

### 风险检查清单

```
请检查以下合同的风险点：

【法律风险】
□ 是否符合相关法律法规？
□ 是否有无效或可撤销条款？
□ 管辖条款是否合理？

【商业风险】
□ 价格是否合理？
□ 付款条件是否对等？
□ 交付时间是否可行？
□ 是否有不公平的排他条款？

【操作风险】
□ 验收标准是否明确？
□ 变更流程是否清晰？
□ 沟通机制是否建立？

【履约风险】
□ 违约责任是否对等？
□ 赔偿上限是否合理？
□ 是否有不可抗力条款？
```

### 风险等级评估

```
请对以下合同条款进行风险评估：

🔴 高风险（必须修改）
- xxx条款：xxx问题，建议xxx

🟡 中风险（建议修改）
- xxx条款：xxx问题，建议xxx

🟢 低风险（可以接受）
- xxx条款：xxx
```

## 7.5 合同对比分析

### 版本对比

```
请对比合同的两个版本：

版本A（原版） vs 版本B（修改版）

1. 列出所有修改点
2. 标注新增条款
3. 标注删除条款
4. 分析修改对各方的影响
5. 给出接受/拒绝建议
```

### 模板对比

```
请对比以下两份合同模板：
1. 共同条款有哪些？
2. 差异条款有哪些？
3. 哪份对甲方更有利？
4. 哪份对乙方更有利？
5. 综合建议采用哪份？
```

## 7.9 专业提示

### ⚠️ 重要提醒

1. **AI不是律师**：AI的建议仅供参考，重要合同请专业律师审查
2. **法规时效**：法律法规会更新，AI的知识可能有滞后
3. **行业差异**：不同行业有不同规范，需要专业判断
4. **保密要求**：上传合同前注意脱敏处理敏感信息
5. **人工复核**：AI生成的合同必须人工复核后才能使用

### 最佳实践

| 场景 | AI角色 | 人工角色 |
|------|--------|---------|
| 初步审查 | 识别风险点 | 判断和决策 |
| 条款对比 | 找出差异 | 评估影响 |
| 起草初稿 | 生成框架 | 补充和调整 |
| 格式检查 | 检查一致性 | 最终确认 |
