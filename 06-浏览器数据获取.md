# 模块六：浏览器在线数据获取与总结

> 🎯 目标：掌握使用AI通过浏览器获取在线数据、进行信息整合和分析总结的能力

---

## 6.1 在线数据获取能力

### AI可以做什么

| 能力 | 说明 | 示例 |
|------|------|------|
| 网页浏览 | 访问指定网页并提取内容 | 查看公司官网信息 |
| 信息搜索 | 搜索特定主题的信息 | 查找行业最新数据 |
| 内容提取 | 从网页中提取结构化信息 | 提取产品参数 |
| 数据对比 | 对比多个网页的信息 | 竞品价格对比 |
| 实时监控 | 定期检查网页变化 | 监控价格变动 |

## 6.2 基本操作流程

### 步骤1：明确目标

```
我需要了解：xxx
我希望获取：xxx
最终用途：xxx
```

### 步骤2：指定来源

```
请帮我访问以下网站获取信息：
1. [xxx.com](https://www.redian.me/) - 官方信息
2. [xxx.com](https://www.stats.gov.cn/xxgk/sjfb/zxfb2020/202405/t20240517_1950399.html)- 行业数据
```

### 步骤3：提取要求

```
从每个页面提取：
- 核心信息
- 关键数据
- 更新日期
- 信息来源
```

### 步骤4：整合总结

```
请整合以上信息，生成：
1. 综合摘要
2. 数据对比表
3. 趋势分析
4. 行动建议
```

## 6.3 常见应用场景

### 场景1：竞品调研

```
请帮我调研以下竞品的最新信息：

1. 访问竞品A官网，提取：
   - 最新产品
   - 价格信息
   - 核心卖点

2. 访问竞品B官网，提取相同信息

3. 对比分析：
   - 功能对比表
   - 价格对比表
   - 优劣势分析
   - 我们的应对策略
```

### 场景2：行业信息收集

```
请帮我收集智能客服行业的最新信息：

1. 搜索并访问行业报告网站
2. 提取最新市场数据：
   - 市场规模
   - 增长率
   - 主要玩家
   - 技术趋势
3. 整理成一份简报
```

### 场景3：新闻舆情监控

```
请帮我查看xxx公司的最新新闻：
1. 访问主要新闻网站
2. 提取近一周相关新闻
3. 按正面/中性/负面分类
4. 总结舆情趋势
```

### 场景4：产品信息查询

```
请帮我查询以下产品的详细信息：
1. 访问产品页面
2. 提取规格参数
3. 查看用户评价
4. 对比官方价格和渠道价格
5. 给出购买建议
```

## 6.5 实操案例

### 案例1：供应商调研

```
请帮我调研3家云服务供应商：

1. 访问各供应商官网
2. 提取以下信息：
   - 产品线和功能

4. 给出推荐建议及理由
```

### 案例2：技术趋势调研

```
请帮我调研AI大模型的最新发展趋势：

   - 最新技术突破
   - 主要厂商动态
   - 应用趋势
   - 政策监管
3. 生成趋势分析报告
```

### 案例3：市场定价调研

```
请帮我调研同类产品的市场定价：

1. 访问5个主要产品页面
2. 记录价格信息
3. 分析定价策略
4. 给出我们的定价建议
```

## 6.7 注意事项

1. **时效性**：在线信息变化快，注意获取时间
2. **准确性**：交叉验证多个来源
3. **版权**：注意信息来源和使用权限
4. **隐私**：避免访问敏感或需要登录的页面
5. **反爬**：部分网站可能限制自动化访问
