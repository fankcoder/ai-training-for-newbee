# 模块六：技能

> 🎯 目标：掌握使用AI的技能来解决问题，与自己制作技能来提升ai的能力

 
## 什么是 OpenClaw「技能」

可以把“技能”理解为：

让 AI 会做一件固定事情的小程序

比如：

自动算利润
生成报价单
分析销售数据


### 技能的基本结构

本质：
👉 AI + 代码 + 输入输出规则

一个最简单的技能，通常包含3个部分：

输入（用户给什么）
处理逻辑（Python代码）
输出（返回什么结果）

你可以用一句话记住：

技能 = 输入 → Python处理 → 输出


### 示例技能：利润计算器


```
帮我实现一个利润计算器，每件商品扣税x%，使用python代码实现，程序文件存放在当前工作空间。
输入：成本，售价，数量，税费x

输出：总收入，总成本，利润
```

### 定义技能

技能名称：利润计算器

技能描述： 用于计算商品销售的总成本、总收入和利润

输入参数：
{
  "cost": "单个成本",
  "price": "单个售价",
  "quantity": "销售数量"
}
